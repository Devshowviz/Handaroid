## 개발목적
- 시각저하가 오신 어르신들이 투약받은 약에 대한 처방을 음성으로 들을 수 있게 지원하고, 약을 검색하여 볼 수 있게 
## 기술개요
- Android Studio - 전반적인 앱 개발
- DeepLearning - CNN (알약구별)
- API - Naver Clova API and Google Cloud TTS
- ANACONDA - Zupiter Notebook (모델적용,데이터크롤링)
